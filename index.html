罗马/FCO		 FCO加油站/租车还车区

停车安全原则：欧洲很多停车场是闸机、摄像头、巡逻或客服值守，不等于有人盯每一台车。意大利与南法段都优先选择酒店车库、官方大型停车场、地下/室内车库、可预约车位；车内绝不放可见物，后备箱也不要在停车场当众打开整理。

8/04 Day8｜刀锋山 + 休斯高原 34欧自驾省钱版

  • 上午睡到自然醒，Santa Cristina / Ortisei早餐和补给；这天不需要很早出门
  • 12:00从Alpstay / Santa Cristina出发，导航Col Raiser缆车站；缆车站停车场先打票，原帖参考4小时约EUR10
  • 12:30-16:30刀锋山半缆车半徒步版：使用已购EUR34/人往返票，Col Raiser缆车约10分钟；下缆车后按Seceda指示牌往左走，约3-4km、1.5-2h到刀锋山打卡点。全程为规划道路，不是野路，坡度不算高，沿途有餐厅
  • 16:30-17:30原路返回下山，到Ortisei或Santa Cristina吃点东西、喝水补能量
  • 17:30-18:00开车去休斯高原，导航Parking Compatsch；17:00前不要上行进入Seiser Alm限行区
  • 18:00-21:00休斯高原日落自驾版：从Parking Compatsch步行往草甸深处，单边约4km、1.5h，平路缓坡，适合慢慢拍。到停车场后不要再往里开，里面是限行区，有罚款风险
  • 21:00以后夏季日落约21点，天黑后谨慎下山，返回Santa Cristina酒店；山路弯多，安全第一
  • 装备提醒软壳/冲锋衣、速干衣、好走的鞋、防晒、饮水；山上风大且晒，体力差或天气差时直接缩短徒步

8/05 Day9｜多洛米蒂 → 卡雷扎湖 → Parma/Piacenza 约430-500km

  • 08:30Alpstay退房，Santa Cristina出发
  • 10:00-11:00Lago di Carezza / Karersee短停拍照，湖边走一小圈
  • 11:00-12:30出山到Bolzano/高速方向，途中加油
  • 12:30-13:30高速服务区午餐
  • 13:30-18:00继续南下，到Parma或Piacenza外围酒店
  • 夜间酒店停车，早点休息，为第二天进入南法留体力

这天车上有全程行李，卡雷扎湖和服务区都只短停，不在停车场整理后备箱。

8/06 Day10｜Parma/Piacenza → Cagnes-sur-Mer 约430-500km

  • 09:00从中转酒店出发，驶向热那亚/意大利海岸
  • 12:30-13:30高速服务区或利古里亚海岸附近午餐
  • 14:00-16:30经意法边境进入法国，开往Cagnes-sur-Mer
  • 16:30-17:30入住Hotel Indigo Cagnes-sur-Mer，确认停车位置
  • 18:00-20:30Cagnes海边/酒店附近轻松晚餐，不再开长途

8/07 Day11｜Èze + Monaco + Menton 约160-190km

  • 08:30从Cagnes-sur-Mer出发，沿A8/海岸方向去Èze
  • 09:45-11:45Èze悬崖小镇，停游客停车场，植物园/海景
  • 11:45-12:30开往Monaco，停地下停车场
  • 12:30-15:00Monaco港口、王宫区、赌场外观，午餐/咖啡
  • 15:00-15:45开往Menton
  • 15:45-17:30Menton彩色老城与海边散步
  • 17:30-19:00返回Cagnes-sur-Mer，车停Hotel Indigo

这天精华很足，晚上回酒店后不再安排远距离活动。摩纳哥只停地下车库，不找路边位。

8/08 Day12｜Antibes + Saint-Paul-de-Vence 约60-100km

  • 09:30从Cagnes-sur-Mer出发去Antibes
  • 10:00-12:30Antibes老城、海边城墙、港口；车停正规停车楼
  • 12:30-14:00午餐，可在Antibes或回Cagnes附近吃
  • 14:30-17:00Saint-Paul-de-Vence艺术小镇，画廊/石头街/咖啡
  • 17:30-19:30返回Cagnes-sur-Mer，海边散步或酒店附近晚餐

如果不想再逛小镇，可把下午改成Cagnes海边休息或Cap d'Antibes短线。

8/09 Day13｜Cagnes-sur-Mer → Pisa → Lucca 约470-520km

  • 08:30Hotel Indigo退房，返回意大利方向
  • 12:30-13:30服务区午餐/加油，尽量停人多明亮区域
  • 14:30-16:00抵达Pisa，停车后去奇迹广场/比萨斜塔短拍照
  • 16:30-17:15开往Lucca
  • 17:30-19:30入住Lucca外围带停车酒店；可轻松走城墙或晚餐

比萨只短停拍照即可；如果要登塔，需另行预约时段。

8/10 Day14｜Lucca → The Mall Firenze → Baccoleno → Sant'Anna 约330-380km

  • 09:00Lucca出发，前往The Mall Firenze
  • 10:30-13:15The Mall Firenze购物村，购物 + 午餐；车停官方停车场
  • 13:15-15:10开往Agriturismo Baccoleno / S湾丝柏树机位
  • 15:10-16:00Baccoleno S湾短停拍照，注意不要挡路或进入私家农地
  • 16:00-17:00开往Pienza附近Monastero Sant'Anna in Camprena
  • 17:00-17:45入住修道院，放行李/购物袋，换衣服休息
  • 17:45-18:20开往Podere Spedalone
  • 18:20-19:50Podere Spedalone周边日落前拍照，争取景观/户外位
  • 20:00Podere Spedalone晚餐开始
  • 21:30左右慢慢开回Sant'Anna修道院休息

8月中旬日落约20:20，19:20-20:20是最适合拍奥尔恰谷金色丘陵的时间。若The Mall逛超时，可跳过Baccoleno，优先保证修道院放行李和20:00晚餐。

8/11 Day15｜Sant'Anna in Camprena → Rome/FCO 约190-230km

  • 08:30-10:00酒店早餐，修道院/田园环境轻松拍照
  • 10:00-11:00可选Pienza短停买羊奶酪/咖啡；不想逛小镇则直接出发
  • 11:00-14:00开回罗马方向，途中服务区休息
  • 14:00-17:00罗马轻松采购/补景点；建议车停酒店、机场或外围车库后打车/地铁进城
  • 17:00-19:00FCO附近加油、还车、入住机场酒店，整理行李

8/12 Day16｜罗马返程 10:40起飞

  • 06:45起床/退房
  • 07:15抵达T3航站楼更稳
  • 07:30-09:00托运、安检、出境
  • 10:40罗马FCO起飞，经伊斯坦布尔返回香港
